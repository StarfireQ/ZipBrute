\# ZipBrute (.NET 9)



A simple console application for brute force breaking password-protected zip files.



---



\## Features



\-  Brute force ZIP file password based on own generated dictionary.
\-  CPU based multi threading.



---



\## Roadmap

\## 1.0.1.0

\-  Using custom dictionary like SecsList


\## 1.0.2.0

\- Support GPU multithreading


---


\### Used technology


\- .NET 9

\- C#

\- SharpZipLib



---

\### Requirements



\- .NET 9 SDK

\- Windows / Linux
